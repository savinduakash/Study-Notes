## ğŸ“¦ Midnight Commander
### Introduction
- Launch: `mc`
- Panels: Left & Right (switch with **Tab**)
- Function keys:
  - **F3** View file
  - **F4** Edit file
  - **F5** Copy
  - **F6** Move/Rename
  - **F7** Create directory
  - **F8** Delete
  - **F9** Top menu
  - **F10** Quit

### File Operations
- Copy/move files between panels
- Rename during copy with masks (`*`, `?`)
- Case conversion with `\u`, `\l`, `\U`, `\L`
